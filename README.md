# Elite Electrical Services Website

A modern, responsive website for an electrical services company. This website features a clean design, smooth animations, and a user-friendly interface.

## Features

- Responsive design that works on all devices
- Modern and clean UI
- Smooth scrolling navigation
- Contact form
- Emergency service button
- Service showcase
- About section with statistics
- Social media integration

## Image Requirements

The website requires the following images to be added to the `images` directory:

1. `logo.png` - Company logo (recommended size: 200x50px)
2. `hero-bg.jpg` - Hero section background (recommended size: 1920x1080px)
3. `residential.png` - Residential services image (recommended size: 600x400px)
4. `commercial.png` - Commercial services image (recommended size: 600x400px)
5. `emergency.png` - Emergency services image (recommended size: 600x400px)

## Setup Instructions

1. Create an `images` directory in the root folder
2. Add all required images to the `images` directory
3. Open `index.html` in a web browser to view the website

## Customization

- Colors can be modified in the `styles.css` file
- Content can be updated in the `index.html` file
- Interactive features can be modified in the `script.js` file

## Browser Support

The website is compatible with:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
